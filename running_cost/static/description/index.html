Stock Move Line - Running Cost

Track Product Cost at the Exact Moment of Stock Movement

Accurate Cost Snapshot & Smart Movement Classification

Key Features

  • ✅ Capture Running Cost at the exact moment of stock validation
  • ✅ Cost is frozen forever and never affected by future price changes
  • ✅ Automatically detect Move Type for each stock operation
  • ✅ Smart classification: Purchase, Sale, Internal, Adjustment, Scrap
  • ✅ Visual color-coded badges for quick identification
  • ✅ Supports multi-step routes (Pick → Pack → Deliver)
  • ✅ Works with Inventory Adjustments and Scrap operations
  • ✅ Fully integrated with Moves History and Detailed Operations
  • ✅ Secure visibility using access groups for cost control

Running Cost Snapshot

The system captures the product cost at the exact moment the transfer is validated. This value remains unchanged forever for audit and reporting accuracy.

Move Type Classification

Each stock move is automatically labeled and displayed as a colored badge.

  • Purchase — Incoming from vendor
  • Sale — Outgoing to customer
  • Internal — Between warehouses/locations
  • Adjustment — Inventory correction
  • Damaged Scrap — Scrapped damaged items
  • Expired Scrap — Scrapped expired items

Moves History View

Running Cost and Move Type are visible directly in the Moves History list.

Detailed Operations

The same information is available inside each transfer under Detailed Operations.

Technical Highlights

  • ✔ Values stored in database (no recomputation)
  • ✔ Written once during _action_done()
  • ✔ Multi-company cost handling supported
  • ✔ Clean and optimized Python logic
  • ✔ Fully compatible with Odoo standard stock flow
